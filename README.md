# 💻 macOS setup

## ✨ Auto script

```bash
❯ chmod +x ../setup.sh
❯ ./setup.sh
```

## 🖼 iTerm2 setup

In Preferences:

- `Profiles > Colors > Import` -> Import the colorscheme from this repo.
- `Keys > Presets... > Natural Text Editing`
- `Text > Cursor > Vertical Bar`

## 👨🏻‍💻 VSCode

Just turn on VSCode settings sync using Github account.
